# Interpretable Machine Learning for Breast Cancer Diagnosis

This project explores interpretable machine learning techniques for breast cancer diagnosis using the Breast Cancer Wisconsin Diagnostic dataset.

## Dataset
- Breast Cancer Wisconsin Diagnostic Dataset (scikit-learn)

## Models Used
- Logistic Regression (interpretable baseline)
- Random Forest (comparison)

## Evaluation Metrics
- ROC-AUC
- Confusion Matrix

## Interpretability
- Permutation Feature Importance

  └── Breast_Cancer_Interpretable_ML.ipynb

Python, scikit-learn, pandas, matplotlib, Jupyter Notebook
